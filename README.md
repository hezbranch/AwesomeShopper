# AwesomeShopper

Working to create the best simple robot for shopping.

## Compiling Agent Code
`javac -cp DIARC.jar Agent.java Goal.java`

## Running Agent Package (Windows OS)
`java -cp .;DIARC.jar Agent Agent `
  
## Running Agent Package (macOS)
`java -cp .:DIARC.jar Agent Agent `
